
# magical-characters

Magical Characters

Magical Characters is a simple web application that displays information about characters from the Harry Potter universe.
The project demonstrates how to fetch data from an external API, dynamically render content using JavaScript, and style a responsive layout with CSS.


Project Overview

This web app allows users to:
	•	View a collection of Harry Potter characters.
	•	Filter characters by their House (Gryffindor, Slytherin, Hufflepuff, Ravenclaw).
	•	Display up to 16 characters at a time.
	•	Replace missing images with a default “Image Not Found” placeholder.
	•	Explore a simple and mobile-friendly interface styled with Flexbox and a Harry Potter-themed font.

Technologies Used
	•	HTML5
	•	CSS3 (Flexbox)
	•	JavaScript (Fetch API)
	•	Harry Potter API → https://hp-api.onrender.com/￼


    Features
	•	Fetch and render character data from API.
	•	Responsive design and hover effects for better user experience.
	•	Dropdown filter by house.
	•	Fallback image for missing character pictures.
	•	Footer displaying developer name and current year.
